# Place and Politics Toronto

Toronto election, census, and population-weighted census-tract interpolation
workflows.

- Project site: https://schoolofcities.github.io/place-and-politics-toronto
- Analysis code and local map viewers:
  `analysis/toronto_election_turnout/`
- Source, processed, audit, and map-ready data:
  `data/toronto_election_turnout/`

See `analysis/toronto_election_turnout/README.md` for setup, rebuild commands,
and the three local Leaflet viewers.
